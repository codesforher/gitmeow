🐈 gitmeow

A tiny pixel cat with big opinions.

Paste your GitHub repository or your code, and gitmeow inspects it, rates it, and leaves a delightfully judgmental review.

## Features

- 🐈 AI-powered code reviews
- 📂 GitHub repository analysis
- ⭐ Repository scorecard
- 🏆 Achievements
- 🎮 Pixel-art animations
- 🐟 Cat moods that change daily

## Tech Stack

- Next.js
- React
- Tailwind CSS
- GitHub API
- OpenAI API
- Vercel
